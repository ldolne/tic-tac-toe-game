Tic Tac Toe

Game's rules

This is a Tic-Tac-Toe game!
Each turn, a player puts their mark in an empty square on the game grid.
The first player to get 3 of their marks in a row wins the game!
If all squares are filled and no player has 3 marks in a row, it's a tie and the game is over.
